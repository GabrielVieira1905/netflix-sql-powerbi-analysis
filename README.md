# Netflix SQL Project

## Objetivo

Analisar o catálogo da Netflix utilizando PostgreSQL para responder perguntas de negócio e praticar SQL.

## Ferramentas

- PostgreSQL
- DBeaver
- GitHub

## Dataset

Netflix Movies and TV Shows Dataset

## Perguntas Respondidas

- Quantos filmes e séries existem?
- Quais países possuem mais títulos?
- Quais anos tiveram mais lançamentos?
- Quais classificações indicativas são mais comuns?

## Principais Insights

- 69,62% do catálogo é composto por filmes.
- 30,38% do catálogo é composto por séries.
- Os Estados Unidos concentram a maior quantidade de títulos.
